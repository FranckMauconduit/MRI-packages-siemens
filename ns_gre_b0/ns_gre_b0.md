# ns_gre_b0 package
> The ns_gre_b0 package is useful to produce b0 maps that can be easily integrated in the [libptx library](https://github.com/neurospin-projects/libptx), a pulse design framework leaded by Vincent Gras. This library is a private repository on GitHub and is only accessible upon request.

## Information

- Software versions: VE12U, XA60

- Accessing the package: either using the Siemens C2P platform (https://webclient.eu.api.teamplay.siemens.com/#/c2p) or using a classic Siemens C2P paperwork.
